_CRT_SECURE_NO_WARNINGS errors

These are some warning generated by Visual Studio in terms of security. To avoid these warnings, you may add _CRT_SECURE_NO_WARNINGS to Project->Properties->C++->Preprocessor->Processor definitions.

Use Visual Studio template

If you want to use the same settings for future projects, you may save current project as a template. And then restart Visual Studio, new project from this template.

Memory limit on Windows

The maximum memory limit for a x86 process in 2GB, while the limit for x64 process can be up to 8TB. Therefore when using OpenCV, if you encounter insufficient memory problem, you may consider compiling your program on x64platform.
