A Study of Intrinsic Image

By soundsilencebisai

Published Aug 17 2014

###Derive Intrinsic Images from Image Sequences (ICCV 2001)

  • Assumption: when derivative filters are applied to natural images, the filter output tend to be sparse.
  • Multiple images from a video: same reflectance image, different illumination
  • $I = R \times L \xrightarrow{\text{log}} i = r + l \xrightarrow{\text{apply filter } f_n} o_n = r_n + l_n$
  • Assume $l_n$ satisfies Laplacian distribution, $P(o_n | r_n) = \frac{1}{Z} \prod_{x,y,t}e ^ {-\beta|o_n(x,y,t) - r_n(x,y)|}$
  • Maximize $p(o_n | r_n)$, get the $r_n$, recover $r$ from $r_n$

###Intrinsic Images Decomposition Using a Local and Global Sparse Representation of Reflectance (CVPR 2011)

  • Assumption: neighboring pixels usually have the same reflectance if their chromaticities are the same or very similar.
  • Use the assumption to restrict the reflectance of similar pixels
  • Constraints on pixels:
    • Smoothness of illumination: minimize $J_{smooth} = \sum||\bigtriangledown^2L||$
    • Reflectance weighting: more smooth at similar pixels $\rightarrow$ larger weights at those locations
    • Reflectance image should be sparse: total variation cost at locations where neighboring chromaticities are significantly different.
  • Minimization energy function

###Intrinsic Image Decomposition with Non-local Texture Cues (CVPR 2008)

  • Assumption: distinct points with the same intensity-normalized texture configuration generally have the same reflectance value.
  • Intensity-normalized texture: use the neighbour pixels’ intensity-normalized color values(chromaticity) to represent texture information
  • Divide pixels into different groups: pixels in the same group have the same reflectance
  • Minimization: weighted refletance difference of neighbours & shading derivatives

###Intrinsic Images Using Optimization (CVPR 2011)

  • Assumption: neighboring pixels in a local window of a single image having similar intensity values should have similar reflectance values.
  • Each pixel’s reflectance value is a weighted sum of its neighbours’ reflectance values: $R_i = \sum_{j \in N(i)} w_{ij} R_j$
  • Energy function: $E(R,s) = \sum_{i \in p} (R_i - \sum_{j \in N(i)} w_{ij} R_j )^2 + \sum_{i \in p}(I_i / s_i - R_i)^2$
  • Minimize energy function

###Intrinsic Images by Clustering (CGF 2012)

  • Assumption: regions of similar chromaticity in the input image have the same reflectance
  • Segment input images according to chromaticity $\rightarrow$ subdivide the segment into clusters $\rightarrow$ refine clusters
  • Segment: CIElab color space, feature $(\beta,a,b)$, k-means
  • Clustering: divide segment into small windows, then merge the window with its neighour windows with smallest chrominance difference
  • Refine clusters: merge small clusters, merge smooth boundaries
  • Constraints:
    • Luminance continuity ($C^0$ continuity): adjacent clusters have the same luminance
    • Cluster of similar reflectance: all clusters coming from the same segment have the same reflectance
  • Solve linear system

###Intrinsic Images in the Wild (SIGGRAPH 2014)

  • Assumption: shading smoothness, reflectance sparseness
  • Energy function: $E(x) = w_p E_p(x) + w_s E_s(x) + w_l E_l(x) $
    • $E_p$: constrain reflectance, encourage pixels with the similar positions, intensity, chromaticity should have similar reflectance.
    • $E_s$: shading smoothness
    • $E_l$: avoid extreme shading values
  • Judgement: WHDR, compare human judgement and algorithm output

###Intrinsic Video and Applications (SIGGRAPH 2014)

  • Initial decomposition for the first frame: cluster based
  • Propagation: for the next frame, for each pixel, find the cluster that it belongs in the previous frame.
    • Determine cluster index through Maximium A Posterior (MAP)
    • Only assign index to pixels with higher confidence
  • Stop condition: number of unassigned pixels in a window $> 80\%$
  • Make use of known reflectance pixels as constraints to determine unknown pixels
  • Backward propagation
  • Coarse-to-fine: determine unassigned pixel values through Possion Blending

###Estimating Intrinsic Component Images using Non-Linear Regression (CVPR 2006)

  • Different from other approaches which focus on reflectance images, this method focuses on recovering shading image first.
  • Non-linear regression: the intrinsic component $c$ of a pixel is evaluated as $r(o)$
    • $o$ is a feature vector generated from a patch of the image
    • $r$ is a non-linear function $o$, supervised training needed
    • Select patches: boosting
  • We can have multiple $r$ with different weights, which could also be learnt.

###User-Assisted Intrinsic Images (SIGGRAPH Asia 2009)

  • Assumption: reflectance variations lie locally in a plane in color space, pixels in planes parallel to the reflectance have constant illumination.
  • There exist $3D$ vector $\mathbf{a}$ such that that $\mathbf{a * R} = 1 \rightarrow \mathbf{a * I = s} $
  • Minimize energy function $\sum(s - \mathbf{a*I})$
    • $\mathbf{a}$ can be represented as a function of $s$
  • User interaction
    • constant reflectance brush: penalize reflectance difference of pixels covered by brushes
    • constant illumination brush: penalize illumination difference of pixels covered by brushes
    • fixed-illumination brush: user can assign a fixed value to illumination

###Color Constancy, Intrinsic Images, and Shape Estimation (ECCV 2012)

  • A unified approach for solving three problems
  • Problem formulation: $\text{minimize}_{(Z,R,L)} g(R) + f(Z) + h(L)$, subject to $I = R + S(Z,L)$
    • R: log-reflectance map
    • Z: depth map
    • L: model of illumination
    • S(Z,L): rendering engine, produce a log-shading image given Z and L
  • $g(R) = w_s g_s(R) + w_e g_e(R) + w_a g_a(R)$
    • $g_s(R)$: priors on local reflectance smoothness, a multivariate Gaussian Scale Mixture placed on the difference between each reflectance pixel and its neighbors
    • $g_e(R)$: priors on global entropy of reflectance, reflectance tends to be sparse, this can be enforced by minimizing entropy of reflectance
    • $g_a(R)$: absolute prior on each pixel in $R$ that prefers some colors over others, addressing color constancy
  • Priors on illumination
    • Spherical-harmonic model
    • Fit a Gaussian distribution on training dataset

##Summary

###Priors

  • Shading smoothness
  • Reflectance sparseness and piecewise constant
  • Texture indicators: same texture, same reflectance

###Methods

  • Optimization
  • Propagation
  • Probability inference
