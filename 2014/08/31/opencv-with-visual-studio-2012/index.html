OpenCV with Visual Studio 2012

By soundsilencebisai

Published Aug 31 2014

  1. Download OpenCV and install.
  2. Add environment variables Path, the value should be <path_to_opencv>\build\x86\vc11\bin (for Visual Studio 2012), or <path_to_opencv>\build\x86\vc10\bin (for Visual Studio 2010);

  3. Create a project in Visual Studio, and set project properties:

    1. In VC++ Directories -> Include Directories, add <path_to_opencv>\build\include <path_to_opencv>\build\include\opencv <path_to_opencv>\build\include\opencv2
    2. In VC++ Directories -> Library Directories, add <path_to_opencv>\build\x86\vc11\lib

  4. Finish
