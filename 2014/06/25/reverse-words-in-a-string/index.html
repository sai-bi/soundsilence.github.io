Reverse Words in a String

By soundsilencebisai

发表于 Jun 25 2014

Given an input string, reverse the string word by word.
Given s = "the sky is blue",
return "blue is sky the".

###題目解析：
給定一個字符串，反轉字符串裏的所有單詞並且輸出。首先，拿到這個問題，不要先急着去寫代碼，先要把要求弄清楚，把題意弄清楚。這三句話並沒有告訴我們所有的信息，比如

  • 什麼叫做一個單詞(word)？
  • 會不會開頭結尾有多餘的空格？
  • 字符串內部會不會有多餘的空格？

這是我們面試時搞清楚的問題，更多的比如要問空間限制，時間限制，需不需要是in place。否則就會是面試官challenge我們的問題。那麼我們現在假設上面問題的答案是:

  • A sequence of non-space characters constitutes a word.
  • Yes. However, your reversed string should not contain leading or trailing spaces.
  • Yes. Reduce them to a single space in the reversed string.

###算法
算法很簡單：找到一個一個的單詞，然後反轉。

或者你可以運用 istringstream 來讀取每一個單詞，這樣會更簡單一點。對於這道題，in place算法不是很方便，而且相比較來說耗費時間會很長，故不適合。
