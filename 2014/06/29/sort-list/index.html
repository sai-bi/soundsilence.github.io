Sort List

By soundsilencebisai

Published Jun 29 2014

Sort a linked list in O(n log n) time using constant space complexity.

###題目解析：
對一個鏈表進行排序，並且時間複雜度要求是$O(n\text{log}n)$，空間複雜度爲$O(1)$。既然空間複雜度爲$O(n\text{log}n)$，那麼我們可選的就只有merge sort，quick sort，heap sort了。對於merge sort，我們的印象中好像它的空間複雜度是$O(n)$，因爲在merge的時候我們需要一個額外的數組，但這是對於數組來說的，對於鏈表就不存在這個問題。因爲數組的插入是需要$O(n)$時間的，而鏈表的插入是常數時間。因此，這道題目選擇merge sort是可以的。當然，這道題目選擇quick sort 也沒問題，只不過它在最壞情況下的複雜度會是$O(n^2)$。

###代碼：
實現的時候要注意的是，因爲merge牽涉到linked list head的改變，而對於改變這樣一個指針，要麼你把新的head作爲返回值，要麼傳進來一個指向head的指針，也就是pointer to pointer。
